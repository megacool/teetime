# teetime
Hackathon project

Getting started
---------------

Setup:

    $ virtualenv venv
    $ . venv/bin/activate
    $ pip install -r requirements.txt

Getting an image from a text:

    $ python teetime/__init__.py <insert sentence to artify here>

Other interfaces will hopefully appear soon.

Running tests
-------------

    $ pip install -r test-requirements.txt
    $ py.test
